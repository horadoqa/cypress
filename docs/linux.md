# Instalando o Node.js no Linux

## Baixando o node.js

```bash
curl -fsSL https://deb.nodesource.com/setup_lts.x | sudo -E bash -
```

## Instalando

```bash
sudo apt-get install -y nodejs
```

## Verificando a versão do Node

```bash
node --version

v20.17.0
```

## Verificando a versão do NPM

```bash
npm --version

v20.17.0
```
